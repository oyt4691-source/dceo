제시해주신 요구사항을 바탕으로, 개인 투자자가 국내외 시장 흐름을 한눈에 파악하고 기업 분석까지 마칠 수 있는 **[주식 통합 현황판 대시보드]**의 PRD를 작성해 드립니다.

[PRD] 통합 주식 현황판 대시보드 (가칭: StockInsight)

버전: 1.0
상태: 초안 (Draft)
작성일: 2024년 5월 22일

1. 프로젝트 개요

목적: 국내/해외 주식 및 ETF 데이터를 통합하여 시각화하고, 뉴스 및 재무 데이터를 제공하여 투자자의 의사결정을 돕는 웹 기반 대시보드 구축.

핵심 가치: 흩어져 있는 투자 정보를 한곳에서(All-in-one) 직관적으로 확인.

2. 타겟 사용자

멀티 투자자: 국내 주식과 미국 주식, ETF(KODEX, Tiger)에 분산 투자하는 사용자.

분석형 투자자: 단순히 가격만 보는 것이 아니라 재무제표와 뉴스 흐름을 함께 파악하려는 사용자.

시각화 선호자: 복잡한 표보다 그래프를 통해 추세를 읽고 싶어 하는 사용자.

3. 상세 기능 요구사항 (Functional Requirements)
3.1. 시장별 현황판 (Asset Overview)

분류 탭 구성:

국내 주식: 코스피, 코스닥 지수 및 주요 종목 실시간/지연 시세.

해외 주식: 나스닥, S&P 500, 다우존스 및 주요 미주 종목(NVDA, AAPL 등).

ETF 전용관: KODEX(삼성자산운용), Tiger(미래에셋) 등 운용사별 필터링 기능 제공.

데이터 포인트: 현재가, 등락률, 거래량, 시가총액 등.

3.2. 멀티 타임프레임 차트 (Advanced Charting)

차트 주기: 일간(Day), 주간(Week), 월간(Month), 연간(Year) 전환 버튼 제공.

기술적 지표: 이동평균선(MA), RSI, MACD 등 기본적인 보조지표 선택 기능.

비교 기능: 특정 종목과 지수(예: 삼성전자 vs KOSPI)를 겹쳐서 비교하는 기능.

3.3. 기업 분석 및 재무제표 (Fundamental Analysis)

100대 기업 리스트: 시가총액 기준 상위 100대 기업 자동 업데이트 및 리스트 제공.

재무제표 조회:

주요 항목: 매출액, 영업이익, 당기순이익, ROE, PER, PBR, 배당수익률.

최근 3개년 연간 데이터 및 최근 4분기 분기 데이터 표기.

관심 종목(Watchlist): 사용자가 검색한 기업을 '즐겨찾기'하여 별도 섹션에서 모니터링.

3.4. 실시간 뉴스 큐레이션 (Stock News)

종합 뉴스: 주요 경제지 실시간 속보 피드.

종목 뉴스: 특정 종목 클릭 시 해당 기업과 관련된 키워드 뉴스 필터링.

감성 분석(Optional): 뉴스 헤드라인의 긍정/부정 수치를 간단한 아이콘으로 표시.

4. UI/UX 디자인 요구사항

레이아웃: 왼쪽 사이드바(메뉴), 중앙 상단(시장 요약), 중앙 하단(차트), 오른쪽 사이드바(뉴스/관심종목)의 3단 그리드 구조.

다크 모드 지원: 장시간 차트 분석을 고려한 다크 모드 기본 탑재.

반응형 웹: PC 브라우저에 최적화하되, 태블릿에서도 조회 가능한 반응형 레이아웃.

5. 기술적 요구사항 및 데이터 소스

Frontend: React.js 또는 Next.js (Dashboard 라이브러리 활용 - 예: Recharts, Chart.js, TradingView Lightweight Charts).

Backend: Node.js 또는 Python (FastAPI).

주식 데이터 API:

국내: 한국투자증권 오픈 API (실시간 시세), 공공데이터포털(금융위 주식시세).

해외/ETF: Yahoo Finance API (yfinance library), Alpha Vantage, Polygon.io.

재무제표: Open DART(금융감독원) API 연동.

뉴스 데이터: 네이버 뉴스 검색 API 또는 Google News RSS 피드.

6. 성공 지표 (Success Metrics)

정보 도달률: 사용자가 종목 검색 후 재무제표 확인까지 걸리는 클릭 횟수 (3회 이내 목표).

체류 시간: 대시보드 페이지에 머무는 평균 시간.

정확성: 실제 시장 데이터와 웹페이지 표시 데이터 간의 오차 범위 최소화.

7. 단계별 개발 로드맵

Phase 1 (MVP): 국내/해외 주요 지수 시세 표시 및 기본 일간/주간 차트 구현.

Phase 2: 100대 기업 재무제표 연동 및 관심 종목 저장 기능 추가.

Phase 3: ETF(KODEX, Tiger) 상세 분류 및 실시간 뉴스 피드 최적화.

Phase 4: 제미나이(Gemini) API를 연동하여 뉴스 요약 및 종목별 투자 리스크 분석 기능 추가.
