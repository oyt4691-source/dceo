# 🚀 dceo: 통합 자산 관리 및 행운 서비스 (LuckyPick & StockInsight KR)

이 저장소는 주식 투자자를 위한 **통합 주식 현황판**과 로또 번호 생성을 도와주는 **럭키픽** 서비스를 포함하고 있습니다. 모든 기능은 설치 없이 웹 브라우저에서 즉시 실행 가능한 단일 HTML 파일로 제작되었습니다.

---

## 📈 1. StockInsight KR (통합 주식 현황판)
`dashboard.html`은 국내 주식 시장에 최적화된 투자 대시보드입니다. 복잡한 수치 대신 직관적인 차트와 실시간 데이터를 제공합니다.

### 🌟 주요 기능 사용법
1. **실시간 시장 요약**: 페이지 상단에서 코스피(KOSPI), 코스닥(KOSDAQ), 그리고 원/달러 환율을 실시간으로 확인하세요.
2. **메인 분석 차트 (종목 변경 가능)**:
   - **퀵 버튼**: 삼성전자, SK하이닉스 등 주요 종목을 클릭 한 번으로 전환합니다.
   - **종목 검색**: 우측 검색창에 종목 코드(예: `035720` - 카카오)를 입력하고 엔터를 누르면 해당 차트가 즉시 로드됩니다.
3. **실시간 뉴스 및 공시**: 우측 타임라인에서 지금 막 올라온 시장 뉴스나 DART(전자공시) 속보를 놓치지 마세요.
4. **시총 상위 100 리스트**: 국내 증시를 이끄는 상위 기업들의 현재가, 등락률, 시가총액 정보를 한눈에 비교할 수 있습니다.

### 💡 주식 초보자를 위한 핵심 용어 가이드
*   **KOSPI (코스피)**: 우리나라를 대표하는 대형 기업(삼성전자, 현대차 등)들이 모여 있는 시장 지수입니다.
*   **KOSDAQ (코스닥)**: 주로 IT, 바이오, 벤처 기업 등 성장성이 높은 중소기업들이 모인 시장입니다.
*   **PER (주가수익비율)**: 기업이 버는 돈에 비해 주가가 얼마나 비싼지 나타냅니다. 낮을수록 저평가된 것으로 보기도 하지만, 업종마다 차이가 있습니다.
*   **시가총액**: (주식 가격 × 발행 주식 수). 해당 기업의 전체 가치이자 '몸무게'라고 생각하면 쉽습니다.

## 🍀 2. LuckyPick (로또 번호 생성기)
`index.html`은 사용자 맞춤형 로또 번호 조합기입니다. 단순 랜덤을 넘어 나만의 행운 번호를 설계하세요.

### 🌟 주요 기능 사용법
1. **원터치 번호 생성**: 메인 버튼을 누르면 로또 공 애니메이션과 함께 6개의 번호가 나타납니다.
2. **조건 설정**:
   - **포함수**: 내가 좋아하는 숫자(최대 3개)를 반드시 포함시켜 번호를 뽑을 수 있습니다.
   - **제외수**: 평소에 잘 나오지 않는다고 생각하는 숫자(최대 10개)를 제거하고 조합할 수 있습니다.
3. **나의 보관함**: 생성된 번호 중 마음에 드는 조합은 브라우저에 저장하여 나중에 다시 확인할 수 있습니다.

## 🛠 기술 스택 (Tech Stack)
*   **Frontend**: HTML5, Tailwind CSS (UI), Vanilla JavaScript (Logic)
*   **Visualization**: Chart.js, TradingView Embedded Widgets (Real-time data)
*   **Data Source**: Google Finance, TradingView, KRX Real-time Feeds

## 🚀 시작하기
1. 이 리포지토리를 클론(Clone)하거나 ZIP으로 다운로드합니다.
2. `dashboard.html` (주식) 또는 `index.html` (로또) 파일을 웹 브라우저(Chrome, Edge 등)로 열기만 하면 끝!
